마무리 작업 회의

음성
 : 음성인식부터, 결과 출력까지 완료
 : 음성관련 모델을 텐서플로우가 아닌 다른 모델 선택으로 감정 분석 정확도 상승 시킬 예정
 : 시연관련해서 웹페이지 이미지 변경 진행 예정

음악
 : 현재 진행되는 장르별 미디파일 추출 완료
 : 학습데이터를 이용해서 마젠타를 이용한 결과 미디파일 생성 완료
 : 쉘스크립트가 아닌 파이썬으로 마젠타 실행하는 내용 실패


마무리 내용 전달

1. 발표자료 작성
: 개인별로 진행한 내용 및 스토리 가이드를 보고 작성하기
2. 소스 정리하기
: 각자 작성한 코드를 모듈화(클래스화)시켜서 깃허브에 올려두기
: 주석달고 소스합치기 시 어려움없이 진행하기 위한 작업내용 (태효가 설명할것)
3. 소스 합치기
: 추후 내가(안장현) 진행
4. 발표 / 시연
: 피피티로 재미를 이끈 후 코드 약간 소개 그리고 실제 시연을 진행한다.

- 발표자료는 내일(3일) 17시에 모여서 경연을 펼치고, 발표자를 선정하고, 추가할 자료를 합쳐서 진행한다.
- 시연은 따로 준비할것
